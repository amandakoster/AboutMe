Something was missing. I wanted to keep building things, though I wanted to influence functionality and user experience along with directing visual communication.
My interest in code began when I really wanted something to be green instead of blue on a Wordpress site. I tweaked the CSS code, it worked, and this piqued my interest. A few years later I was managing a website for a hackathon project, wanted our site to have a cool favicon, and to figure it out myself. I found the code and directions on Wikipedia, entered the code, saved the HTML file and hit refresh. When the red favicon appeared I was stunned. It felt like I had performed magic and wanted more.
